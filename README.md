# Premium Portfolio Website

A modern, animated personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS. This project showcases a polished UI with sections for about, projects, services, experience, and contact information.

## ✨ Features

- Responsive one-page portfolio layout
- Smooth scroll navigation and mobile menu
- Animated hero and section transitions
- Downloadable resume call-to-action
- Modern glassmorphism and gradient styling
- Built with reusable UI components and icons

## 🛠️ Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Motion
- Lucide React

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the local preview in your browser:
   ```text
   http://localhost:5173
   ```

## 🧱 Build for Production

```bash
npm run build
```
